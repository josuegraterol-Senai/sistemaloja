# Sistema de Loja de Roupas Online 👕

Este é um projeto de uma loja de roupas funcional construído com **HTML**, **JavaScript** e **Firebase**.

## 🚀 Funcionalidades
- **Catálogo de Produtos:** Exibição dinâmica via Firestore.
- **Autenticação:** Login com Email/Senha e Google.
- **Painel Admin:** Cadastro de produtos com upload de fotos (Storage).
- **Carrinho e Favoritos:** Persistência de dados por usuário.

## 🛠️ Tecnologias
- Front-end: HTML5, CSS3, Vanilla JS.
- Backend-as-a-Service: Firebase (Auth, Firestore, Storage).

## 📦 Como rodar
1. Clone o repositório.
2. Abra o `index.html` usando um servidor local (ex: Live Server do VS Code).
3. Certifique-se de que o domínio `localhost` está autorizado no seu Console do Firebase.
